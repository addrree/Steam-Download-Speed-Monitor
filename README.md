# Steam-Download-Speed-Monitor
Cкрипт, который в фоне/консоли отслеживает скорость загрузки игры в Steam.
Каждую минуту (5 раз) выводит:
- какая игра сейчас скачивается (name)
- текущую **скорость загрузки** (download speed) из логов Steam
- статус `downloading` / `paused/idle`

## Как работает
1. Находит Steam независимо от места установки:
   - **Windows**: путь из реестра `HKCU\Software\Valve\Steam` (`SteamPath/InstallPath`)
   - Linux/macOS: стандартные директории (fallback)
2. Определяет активные загрузки по наличию папок:
   - `steamapps/downloading/<appid>`
   - чтобы не ловить “хвосты” старых загрузок, считаем активными только те папки, которые **менялись недавно** (`mtime`).
3. Скорость загрузки берётся из логов Steam:
   - `Steam/logs/download_log.txt`
   - `Steam/logs/content_log.txt`
4. Логи читаются **инкрементально**:
   - при запуске ставим указатель чтения **в конец файла** (историю не читаем)
   - дальше читаем только **новые дописанные строки**
   - если лог давно не обновлялся → считаем скорость = `0`

## Запуск
Требуется Python 3.10+.

```bash
python steam_download_monitor.py
